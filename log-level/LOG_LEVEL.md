# Log Level

Given an integer and boolean flag:
- Return the label of the log line as an atom
- Determine who needs to be alerted


| Log Code              | Log Label  | Supported in Legacy Apps? |
|----------------------|-----------|--------------------------|
| 0                   | trace     | no                       |
| 1                   | debug     | yes                      |
| 2                   | info      | yes                      |
| 3                   | warning   | yes                      |
| 4                   | error     | yes                      |
| 5                   | fatal     | no                       |
| other / not supported | unknown   | -                        |
